# Restaurador de Fotos con Python

Aplicación en Python para restaurar y mejorar fotografías antiguas o dañadas.

## Características

- Mejora de nitidez
- Reducción de ruido
- Corrección de color
- Restauración de detalles

## Instalación

Clona el repositorio:

git clone https://github.com/leonidasu2/rest.git

Instala dependencias:

pip install -r requirements.txt

## Uso

python app.py
